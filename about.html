About Project

U-GRANT is a Loan processing system that allows any individual or Organization to sell/buy loan without bank dependency. Manual loan can be difficult to obtain, because bank require comprehensive information about clients’ loans in an efficient and timely manner. Information about the client are stored in their specific branches, it takes unnecessary efforts to copy and send documentation to headquarters making this very time-consuming and inefficient for loan processing. Realizing how inefficient this is, we decided it was time to digitize the concept of loan processing. This however as to be done in stages, to see how productivity increased.
